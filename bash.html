Bash Love Letter

Portal • Bash • Git • Github

You’ve touched me in ways I never knew I could be…

Pwd: parent working directory
Ls: list what is inside directory
Cd: change directory
…: return to previous directory
mkdir: make a new directory/folder
touch: make new file
nano: simple text editor

Bash, you make me feel like a true coder. I have never looked more bad ass in a coffee shop then I do with you. You simplify my life and save me seconds. In the terminal I was always lost, but because of you now I am found.

Bash is a shell for the computer or a command language interpretor, used through your computer terminal. After opening the terminal you can do anything you would normally use through the GUI (graphical user interfacing). After opening the terminal you can use any of the commands shown above to move thorugh your computer files, create new files, edit files, launch applications and much more.
