Hubba Hubba I git lucky with you

Git diff: Shows changed
Git push: upload commits to git server from local server
Git pull: Shows if there are any changed on git server
Git merge: downloads file changed to local server

Uploading to you saves me. Having so many versions in one place gives me the ability to always change knowing you will always be there for me. You have saved my projects and my grades many times. I don't know what I would do without you.

Github is a web based GIT/version control repository. It allows you to collaborate with individuals and to have the safety of previous versions that you can resort back to. It is accessed through the terminal using git commands and logging into github.com.
